para um layout responsivo

Using in styled component:
```js
import {BreakAt, BreakpointSizes} from './Breakpoint'

const Title = Styled.h1`
    font-size: 1em;
${BreakAt(BreakpointSizes.md)}{
     font-size: 2em;
}
`
```
